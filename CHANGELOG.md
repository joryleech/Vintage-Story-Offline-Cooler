# Changelog

All notable changes to Offline Cooler are documented here.

The project uses [Semantic Versioning](https://semver.org/).

## [1.0.0] - 2026-07-20

### Added

- Fat-sealed storage vessels in every standard fired-clay color.
- Shapeless recipe using one fired storage vessel and one rendered fat.
- Persistent owner UID and player-name metadata.
- Owner labels in placed-block information and held-item tooltips.
- Offline spoilage prevention based on the vessel owner's connection state.
- Offline-time accounting across chunk unloads and server restarts.
- Twelve storage slots with vanilla storage-vessel preservation bonuses.
